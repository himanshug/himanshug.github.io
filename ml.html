Learning to do Machine Learning

There is so much data all around. Companies like google, facebook, amazon used data and machine learning(and information retrieval) to come up with various cool features that gave them unique competitive advantages. Netflix had a million dollar price competition to improve their prediction algorithms. Somehow motivated with all of this, I became excited with the domain of machine learning and started exploring. I got a lot of guidance from following 2 resources:

Ravi Mohan's Learning about Machine Learning post

Bradford Cross's Learning about Machine Learning post

From above and a little peeking into Pattern Recognition and Machine Learning it became clear that I can not make any serious progress without actually having firm grounding in how to prove stuff, discrete maths, probability, statistics, calculus, linear algebra etc.

So I thought I will spend a year to cover all the prerequisites :). I don't know where all of this is going to take me but it sure is fun. So, Here is my ongoing progress...

  1. I worked through and completed pretty much every exercise in How to prove it to learn to think like mathematicians. All my progress with this book is noted in these posts on my blog.
  2. For discrete mathematics, I followed all the video lectures from this course and worked through pretty much every problem set. Again, these posts contain all my progress.
  3. For probability, I worked through Introduction to Probability and finished problem sets in introductory probability course at MIT. Again, the progress is noted on these posts.
  4. Just to get an overview of machine learning algorithms, I followed video lectures provided in stanford machine learning course. Also, tried to captured the high level learnings here.
  5. To get accustomed with basic AI stuff, I implemented many algorithms(in Scala) from Artificial Intelligence: A Modern Approach. Here is the code.
  6. For basic statistics, I worked through some chapters in Mathematical Statistics and Data Analysis and did the problem sets from MIT-Statistics for Applications course. Again, the progress is here.
  7. Now a days, I am working my way through the overwhelming amount of maths in Pattern Recognition and Machine Learning
